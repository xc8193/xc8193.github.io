Montreal Metro Pink Line

Speculative GIS Study and Urban Impact

Website and Css Wildfire Project A small program linked to SQL Metro Speculation project

Project Summary

This GIS-based research explores the potential implementation of a new Pink Line in the Montreal metro system. Through spatial analysis and data layering using QGIS, the project evaluates how the Pink Line might affect accessibility, population distribution, and transit equity across key neighborhoods in Montreal.

Tools & Data

  • QGIS for spatial analysis and visualization
  • STM datasets for metro lines and stops
  • Census tract shapefiles for demographic overlays
  • Custom georeferenced Pink Line route
