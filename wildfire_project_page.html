Wildfire Vegetation Loss

Satellite-Based NDVI Analysis (2015–2017)

Website and Css Wildfire Project A small program linked to SQL Metro Speculation project

Overview

This project used Landsat imagery and TerrSet’s IDRISI tools to analyze vegetation loss caused by wildfires across multiple years. By comparing NDVI and NBR indices from 2015, 2016, and 2017, burn severity was classified and mapped using supervised classification, thresholding, and change detection techniques.

Methods & Tools

  • Landsat 8 satellite imagery (2015–2017)
  • NDVI and NBR calculations for vegetation and burn severity
  • Image processing with TerrSet and ArcGIS
  • Supervised classification, reclassification, mosaicking
