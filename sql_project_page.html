A database-integrated desktop application

Website and Css Wildfire Project A small program linked to SQL Metro Speculation project

Overview

This application was developed to demonstrate how a local program can interact with a structured SQL database. The interface is designed for clarity and usability, while the backend handles live data operations.

Features

  • Connects to a SQL database for real-time data interaction
  • User input fields with validation
  • Supports Create, Read, Update, Delete operations
  • Error handling and basic logging
